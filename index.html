10year AUS Open Matches

Description:

The main idea of this visualizations is to analyse how each country is performing on whole of 10 years and then dig in to performance in each year of a particular country.

The first Visualization(bar chart) gives an overview of the total number of games won vs lost in stacked bar graph, in which length of green part signifies number of wins and length of orange bar signifies number of games lost.

On selection of any of the counties in overview chart populates the detailed view chart(line graph) with the selected country's performance in each year in a time series line chart. Similar to first chart this has two lines the green line signifies wins and orange for loses. By default when overview chart is loaded detail view chart is populated with first country in the overview chart.

Example: On click of 'RUS' bar in overview chart the detail view shows the Russia's year wise performance in detail view chart

Extra Credit: Many counties from different continents have participated in AUS open so displaying all countries data in a single chart might make it look clumsy. So, filtering countries based on continent is provided on top of the overview chart. Example: When user is interested in Asian countries performance then he need to select Asia from select drop-down and both charts reflects data of Asain countries.
